# What is firewall?
A firewall is a network security device that monitors, filters, and controls incoming and outgoing network traffic based on a defined set of security rules.

# Types of firewall
Packet-filtering firewall: Filters packets based on IP addresses, ports, and protocols.
Stateful inspection firewall: Tracks the state of active connections and makes decisions based on connection context.
Application-layer (proxy) firewall: A proxy firewall (also called an application-level gateway or application proxy firewall) is a type of firewall that acts as an intermediary between a client and a destination server. Instead of allowing a direct connection, the proxy firewall terminates the client's connection, inspects the request, and then establishes a separate connection to the destination on the client's behalf.
Next-Generation Firewall (NGFW): Combines traditional firewall functions with deep packet inspection, intrusion prevention, application awareness, and threat intelligence.
